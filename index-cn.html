声明式特征工程

数据科学家可以使用熟悉的Python，以声明式的方式定义特征转换逻辑。平台自动将其编译成高效的数据处理任务，并在后台调度执行，无需手动编写和维护ETL脚本。


@batch_feature_view(
  sources=[transactions],
  entities=[user],
  mode='spark_sql',
  ttl='30d',
  schema=[Field('user_id', Int32), Field('total_spend', Float32)]
)
def user_total_spend(transactions_df):
    return f"""
        SELECT
            user_id,
            SUM(amount) as total_spend,
            CURRENT_TIMESTAMP() as timestamp
        FROM {transactions_df}
        GROUP BY user_id
    """

简洁的Python定义取代了数百行复杂的ETL代码。
